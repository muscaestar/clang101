## 寄存器专题
### 实现两个数字的交换 swapTwoNum
```c
int a = 10, b = 100, ret;
if (a > b) {
	ret = a;
}

ret = b;
```

### 循环结构 loopstruct
```c
int a = 10;
while(a > 0) {
	a--;
}

int b = 5;
do {
    b--;
} while(b > 0);
    
for (int c = 0; c < 8; c++) {
    b++;
}
```

### CaseSwitch caseswitch
```c
int a = 10;
switch(a) {
    case 1: break;
    case 10: a++;
}
```
